# Configuração Avançada do Aquecimento

Para usuários experientes que desejam máximo controle sobre o processo de aquecimento, oferecemos configurações avançadas que permitem personalizar cada aspecto do sistema.

## Parâmetros de Temporização

### Intervalos Entre Mensagens

Configure intervalos mais precisos para simular comportamento humano:

```json
{
  "intervalos": {
    "minimo": "30m",
    "maximo": "4h",
    "picos_atividade": {
      "manha": "15m-45m",
      "tarde": "20m-1h",
      "noite": "45m-2h"
    }
  }
}
```

### Horários de Atividade

Personalize quando suas instâncias devem estar ativas:

- **Horário Comercial**: 8h às 18h
- **Horário Estendido**: 7h às 22h
- **24 Horas**: Atividade contínua (não recomendado)
- **Personalizado**: Defina seus próprios horários

### Padrões Semanais

Configure diferentes intensidades por dia da semana:

- **Segunda a Sexta**: 100% da atividade
- **Sábado**: 70% da atividade
- **Domingo**: 50% da atividade

## Tipos de Mensagem Avançados

### Distribuição de Conteúdo

Configure a porcentagem de cada tipo:

- **Texto Simples**: 60%
- **Texto com Emoji**: 25%
- **Mídia (Imagem/GIF)**: 10%
- **Áudio**: 3%
- **Stickers**: 2%

### Templates Personalizados

Crie seus próprios templates de mensagem:

```javascript
// Exemplo de template personalizado
const templates = {
  saudacoes: [
    "Oi {nome}, tudo bem?",
    "E aí {nome}, como está?",
    "Olá! Como foi seu dia?",
  ],
  despedidas: ["Até mais!", "Falou, {nome}!", "Tchau, até a próxima!"],
};
```

### Contexto Inteligente

Configure respostas baseadas em contexto:

- **Horário**: Mensagens diferentes para manhã/tarde/noite
- **Dia da Semana**: Conteúdo específico para cada dia
- **Histórico**: Referências a conversas anteriores
- **Sazonalidade**: Mensagens temáticas (feriados, eventos)

## Configurações de Números Externos

### Categorização de Contatos

Organize seus números externos por categoria:

#### Números VIP (Resposta Garantida)

- Membros da equipe
- Contas de teste dedicadas
- Números com resposta automática

#### Números Regulares

- Contatos pessoais
- Parceiros de negócio
- Clientes que autorizaram

#### Números de Backup

- Números secundários
- Contas antigas
- Contatos esporádicos

### Rotação Inteligente

Configure como os números são utilizados:

- **Sequencial**: Um por vez, em ordem
- **Aleatório**: Seleção randômica
- **Baseado em Resposta**: Prioriza quem responde mais
- **Balanceado**: Distribui igualmente

### Configuração de Grupos

Para mensagens em grupos:

```json
{
  "grupos": {
    "criar_automaticamente": true,
    "tamanho_minimo": 3,
    "tamanho_maximo": 8,
    "atividade_por_grupo": "2-5 mensagens/dia",
    "tipos_permitidos": ["texto", "midia", "reacoes"]
  }
}
```

## Algoritmos de Naturalidade

### Variação de Velocidade

Simule padrões humanos de digitação:

- **Velocidade Base**: 40-60 caracteres por minuto
- **Variação**: ±20% baseada no comprimento
- **Pausas**: Simula pensamento entre frases
- **Correções**: Ocasionalmente "corrige" erros

### Padrões de Resposta

Configure como as instâncias respondem:

#### Tempo de Resposta

- **Imediato**: 0-30 segundos (5%)
- **Rápido**: 30 segundos - 2 minutos (25%)
- **Normal**: 2-15 minutos (50%)
- **Lento**: 15 minutos - 2 horas (20%)

#### Taxa de Resposta

- **Mensagens Diretas**: 85-95%
- **Mensagens em Grupo**: 30-50%
- **Mensagens de Mídia**: 60-70%
- **Perguntas**: 90-95%

### Simulação de Estados

Configure diferentes "humores" da conta:

- **Ativo**: Responde rapidamente, inicia conversas
- **Ocupado**: Respostas curtas, menos frequentes
- **Ausente**: Não responde por períodos
- **Social**: Mais ativo em grupos, usa mais emojis

## Configurações de Segurança

### Limites de Proteção

Configure limites para evitar banimentos:

```json
{
  "limites_diarios": {
    "mensagens_enviadas": 50,
    "mensagens_recebidas": 100,
    "grupos_criados": 2,
    "contatos_adicionados": 10
  },
  "limites_horarios": {
    "mensagens_por_hora": 8,
    "intervalo_minimo": "5m"
  }
}
```

### Detecção de Anomalias

Sistema inteligente que detecta padrões suspeitos:

- **Velocidade Excessiva**: Para automaticamente se detectar
- **Conteúdo Repetitivo**: Varia mensagens automaticamente
- **Horários Suspeitos**: Evita atividade em horários incomuns
- **Padrões Robóticos**: Adiciona aleatoriedade extra

### Modo Stealth

Configurações ultra-conservadoras para contas sensíveis:

- **Velocidade**: 50% mais lenta
- **Variação**: 200% mais aleatória
- **Conteúdo**: Apenas texto simples
- **Horários**: Apenas horário comercial

## Integração com APIs Externas

### Geração de Conteúdo

Integre com serviços de IA para conteúdo dinâmico:

```javascript
// Exemplo de integração com OpenAI
const gerarMensagem = async (contexto) => {
  const response = await openai.chat.completions.create({
    model: "gpt-3.5-turbo",
    messages: [
      {
        role: "user",
        content: `Gere uma mensagem casual para WhatsApp no contexto: ${contexto}`,
      },
    ],
    max_tokens: 50,
  });

  return response.choices[0].message.content;
};
```

### Validação de Números

Integre com serviços de validação:

- **Verificação de Existência**: Confirma se número existe
- **Status WhatsApp**: Verifica se tem WhatsApp ativo
- **Histórico de Spam**: Evita números problemáticos
- **Geolocalização**: Respeita fusos horários

### Análise de Sentimento

Configure respostas baseadas no sentimento:

- **Positivo**: Respostas mais entusiasmadas
- **Neutro**: Respostas padrão
- **Negativo**: Respostas mais cuidadosas
- **Pergunta**: Respostas mais informativas

### Alertas Customizados

Configure alertas específicos:

- **Taxa de Entrega < 90%**: Alerta amarelo

- **Diário**: Resumo das atividades

## Configurações de Backup e Recuperação

### Backup Automático

Configure backups das configurações:

- **Frequência**: Diário, semanal ou mensal
- **Retenção**: Quantos backups manter
- **Localização**: Local ou nuvem
- **Criptografia**: Proteger dados sensíveis

### Recuperação de Desastres

Plano para situações críticas:

- **Instância Banida**: Protocolo de contenção
- **Múltiplos Banimentos**: Parada de emergência
- **Falha do Sistema**: Recuperação automática
- **Perda de Dados**: Restauração de backup

## Otimização de Performance

### Configurações de Sistema

Otimize o desempenho do aquecimento:

```json
{
  "performance": {
    "threads_simultaneas": 4,
    "cache_mensagens": true,
    "compressao_logs": true,
    "limpeza_automatica": "7 dias"
  }
}
```

### Balanceamento de Carga

Para múltiplas instâncias:

- **Distribuição Uniforme**: Mesma carga para todas
- **Baseada em Capacidade**: Mais carga para instâncias mais fortes
- **Prioridade**: Algumas instâncias têm prioridade
- **Adaptativa**: Ajusta baseado na performance
